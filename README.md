# Sahamati-Ilve-prototype

A small prototype for the Sahamati-Ilve integration/UI components. This repository contains the HTML prototype demonstrating the UI layout and basic interactions used during early design and testing.

## Contents
- `index.html` — main prototype page (HTML only)
- `assets/` — images, styles, and other static assets used by the prototype

## Purpose
This repository is intended to:
- Quickly demonstrate UI/UX ideas
- Validate layout and interaction flows
- Share a lightweight prototype with collaborators

## How to run
1. Clone the repository:

   git clone https://github.com/shreyansh-2002/Sahamati-Ilve-prototype.git
2. Open `index.html` in your browser (double-click or serve via a static server):

   - Using Python 3 server:
     ```bash
     python -m http.server 8000
     # then open http://localhost:8000 in your browser
     ```
